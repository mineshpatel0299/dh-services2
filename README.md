# Interactive Selector 2

A standalone Next.js project featuring an interactive technology sectors selector component.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Interactive hoverable sections showcasing different technology sectors
- Smooth animations and transitions
- Responsive design
- Built with Next.js 15, React 18, and Tailwind CSS

## Build for Production

```bash
npm run build
npm start
```

## Deploy

This project can be easily deployed to Vercel, Netlify, or any other Next.js hosting platform.

For Vercel deployment:
```bash
vercel
```
